# KamiLog
Track your Anime wherever and whenever.

- A personal project to track your favorite shows and be updated with the hottest shows.

---

## Release Version 1
- Anime Database
  - Browse any anime
  - Check Info and episode releases
  - Be updated with anime release schedule
  - Search animes that you want to see
 
---

## Upcoming Features
- **Anime Tracking**
  - Create custom lists
  - Track shows that you have completed, watching, planning to watch, or dropped.
  - Log In using your account to access your list
  - Track which episode have you seen or completed
 
- **Manga Tracking**
  - Do everything but with Mangas

- **Forums**
  - Engage in discussion with other users about your favorite shows
